# 📊 Excel From the Beginning – Project Portfolio

This repository contains the five Excel projects completed from the freeCodeCamp course **"Microsoft Excel Tutorial for Beginners – Full Course"**. Each project focuses on real-world business scenarios while developing essential Excel skills in calculations, reporting, data management, and analysis.

---

# 📁 Projects

## 1. Payroll Calculator

### Description
A payroll worksheet that calculates employee compensation, overtime pay, taxes, and total earnings. This project introduces spreadsheet fundamentals and formula creation.

### Excel Functions Used
- `SUM()` – Calculate total hours and earnings
- `IF()` – Determine overtime eligibility
- Basic arithmetic operators (`+`, `-`, `*`, `/`)
- Absolute references 

### Excel Skills Used
- Data entry
- Formula creation
- Relative vs Absolute references
- Currency formatting
- Autofill
- Spreadsheet organization
